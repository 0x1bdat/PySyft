# hagrid

Use this cli to deploy PyGrid Domain and Network nodes on your local machine.

A Hagrid is a HAppy GRID!

## Installation
```bash
pip install -r requirements.txt
pip install -e .

# NOTE: don't use 'python setup.py install' or 'pip install .'
```
